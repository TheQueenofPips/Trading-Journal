Non-negotiables — read before touching the keyboard
These rules exist because you've already learned the hard lessons.
R1One target, one exit. When an account hits its weekly target — withdraw it, close that platform. No extra trades.
R2Hard daily stop-out = 50% of weekly target. Hit it → computers off, no re-entries, no exceptions.
R3No impulse entries. If it wasn't in your morning plan — run the FOMO checklist first. Every time.
R43 consecutive losers = session over. Walk away. Thailand is still there tomorrow.
R5Never move a stop in the wrong direction. Once set — it does not move wider. Ever.
R6Each account is a separate business. A bad day on one account does NOT give permission to trade harder on another.
R7Survival mode = no-trade day. If you feel you NEED money today — close the laptop. Come back tomorrow.
R8The math already works. You only need consistent execution. Don't blow it chasing more.
