The 90-second pause
1:30

Start the timer. Step back from the screen. Breathe. Answer every question below honestly. If the timer expires and conviction remains — it might be real. FOMO fades in 90 seconds. Real setups don't.

FOMO checklist
My rules
Non-negotiables — read before touching the keyboard
These rules exist because you've already learned the hard lessons. Don't relearn them.
R1 One target, one exit. When an account hits $1,900 for the week — withdraw it, close that platform. No extra trades. The account is done for the week.
R2 Hard daily stop-out = $950. That's 50% of your weekly target. Hit it on any account → computers off, no re-entries, no exceptions. Not "one more trade."
R3 No impulse entries. If it wasn't in your morning plan or isn't a textbook reactive signal — it's FOMO. Run the FOMO checklist. Take the 90-second pause. Every time.
R4 3 consecutive losers = session over. Consecutive losses signal you're off your game. Walk away. Thailand is still there tomorrow. The market will still be there tomorrow.
R5 Never move a stop in the wrong direction. If you placed it, it's sacred. Stop-hunting your own stop losses is how accounts blow. Once set — it does not move wider.
R6 Each account is a separate business. A bad day on Apex #1 does NOT give permission to trade harder on Apex #2 to compensate. They are independent. Treat them that way.
R7 Survival mode = no-trade day. If you feel you NEED to make money today — that is the exact psychological state that blows accounts. Recognize it. Close the laptop. Come back tomorrow.
R8 The math already works. 8 accounts × $1,900 = $15,200/wk. You need $3,125/wk to hit $50K in 16 weeks. That's 20% of what's available. You only need to not blow it.
