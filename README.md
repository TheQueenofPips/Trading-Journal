# Discipline First — Trading Journal

A personal prop trading journal built for focus, discipline, and accountability.

## Features

- **Dashboard** — weekly target tracker, discipline streak, position collar
- **Pre-trade ritual** — emotional state gate, market plan, saved ritual history
- **Trade log** — full trade cards with entry type, stop discipline, FOMO score, notes
- **FOMO check** — 90-second pause timer + 8-point checklist
- **Account manager** — add/remove/pause any prop firm account dynamically
- **Rules** — 8 core rules + custom rules
- **Thailand goal** — $50,000 savings tracker with week-by-week plan
- **Mindset journal** — 7 entry types, mood tracking, guided prompts, search + filter

## Usage

Open `index.html` in any browser. All data saves automatically to browser localStorage.

## Deployment

This is a single-file app — no build step, no dependencies, no server needed.
Just open the HTML file or host it on GitHub Pages.
